# Changelog

All notable changes to this project will be documented in this file.
This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-07-21

### Added
- Initial public release of Earth Guardian AI
- Design system: dark atmospheric theme, glassmorphism panels, telemetry
  ticker, custom radial gauge, custom SVG world map with animated hotspots
- 14 intelligence modules + Executive Dashboard + Landing page:
  Climate Intelligence, Disaster Prediction, Air Quality, Water Security,
  Forest & Biodiversity, Renewable Energy, Carbon Footprint Analyzer,
  Smart Agriculture, Ocean Protection, Environmental Risk Scanner,
  Smart Cities, Learning Hub, Volunteer & NGO Network, Research Center
- Fully interactive carbon footprint calculator with live gauge/pie updates
- Realistic mock dataset covering 14 cities, 8 active disaster events,
  16-year global climate trend, reservoirs, energy mix, forests, oceans,
  smart-city benchmarks, NGOs, and learning modules
- Full documentation set: README, project overview, technical architecture,
  business model, impact report, roadmap, pitch deck, university portfolio
  notes, research paper outline, awards & competitions guide, GitHub metadata
- MIT License, Contributing guide

### Known Limitations
- No backend; all data is mock data (disclosed throughout documentation)
- No authentication or multi-tenant support
- Single JS bundle (~765KB); route-based code-splitting planned for v1.1
